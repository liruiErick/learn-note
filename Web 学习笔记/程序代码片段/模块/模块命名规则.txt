1) 模块方法：文件名与模块名都为驼峰命名法，首单词的首字母小写，返回值为模块方法。

2) 模块类：文件名与模块名都为驼峰命名法，首单词的首字母大写，返回值为模块类。

3) 自动执行模块：文件名与模块名全部小写，无返回值。

4) jQuery 模块：文件名与模块名以 jquery. 开头，文件名与模块名都为驼峰命名法，首单词的首字母小写，返回值为 jQuery 或者 $。


## 语义化版本号
版本格式：主版号.次版号.修订号，版号递增规则如下：
- 主版号：当你做了不相容的 API 修改，
- 次版号：当你做了向下相容的功能性新增，
- 修订号：当你做了向下相容的问题修正。
- 先行版号及版本编译资讯可以加到「主版号.次版号.修订号」的后面，作为延伸。
