* Container 对象的 hitTest 方法失效问题

原因：
在查看源代码后发现，在 Container 对象的 hitTest 方法中，有一条很奇怪的 if 语句，该语句会判断 Container 对象中所有非 Container 对象的子对象是否注册了鼠标相关事件，如果没有注册，则不响应碰撞检测。
其中检测鼠标事件的函数为 _hasMouseEventListener()，它的源码中显示，只有被检测对象注册了鼠标事件，或者定义了 cursor 指针样式，才会返回 true。

解决方案：
1、遍历该 Container 对象，为所有非 Container 对象的子对象设置 cursor 指针样式为 "auto" 或其他不影响正常外观的指针样式（测试发现输入任意非正确样式值的字符都是可行的）。
2、遍历该 Container 对象，为所有非 Container 对象的子对象注册一个鼠标事件。




* Chrome 浏览器下部分滤镜效果无效问题

原因：
此问题是由于浏览器的跨域安全设置引起，本地访问页面时部分滤镜效果无法正常工作。如果在服务器上运行，则一切正常。
